levels/level1.js

Defines and initializes the configuration of "level1" in the game. This includes spawning enemies, collectible coins, throwable bottles, clouds, and background layers. The level is dynamically populated with randomized positions for better replayability.

scripts/game.js

Main script to handle the start screen, game controls, fullscreen toggle, sound, and game state. The script sets up event listeners for user interactions and controls the flow of the game. It manages the visibility of the start screen, settings, and credits, as well as handles game start, restart, and sound functionalities.
